DD_API_KEY=2f79eb90fab7d0db98f3da50e106b21d DD_SITE="us5.datadoghq.com" bash -c "$(curl -L https://s3.amazonaws.com/dd-agent/scripts/install_script_agent7.sh)"
systemctl start datadog-agent

Once completed, check here: https://us5.datadoghq.com/
Go to Dashboard > Dashboard List > All Hosts > Double click on your linux host

Enable Process Monitor:
vim /etc/datadog-agent/datadog.yaml

Add below lines
1366 process_config:
1367     process_collection:
1368         enabled: "true"

without line numbers
process_config:
    process_collection:
        enabled: "true"

Save and restart agent.
systemctl restart datadog-agent

Once enabled,
Monitors > Monitors List > New Monitor > Live Processes
Select processes
	Search processes: command:httd
Set alert conditions
	Alert threshold: < 1
	Warning threshold: < 3
Select Simple Alert
